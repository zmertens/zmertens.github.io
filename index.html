PROJECTS

The following is a summary list of some of my once and future pet projects:

  1. MazeGameCpp My attempt at making a first-person shooter from scratch. Inspiration came from Benny QBD's Java Wolfenstein clone I also added some Phong shading and Perlin noise and some sounds and music. I never really could get past that point once I had those features working. I thought of trying to add A-star pathfinding and automated level generation but I lost interest. Overall, this was a good learning experience and a fun one-man-wolf-project. This lecture at Digipen has a a real good take on game development processes.
  2. checkers This started as a school project for an *Introduction to Programming Languages, CSE240* course at **Arizona State University**. The idea was to create the Checkers logic in C++, and for extra credit, add a GUI. We never got to adding a real GUI, but afterwords I kept working on it and used SFML to create some graphics. It helped me land an internship later on which was great.
  3. seq & destroy I helped work on this project for a community outreach event at the BioDesign Institute at Arizona State University. The idea was to make a bubble-shooter-like game that emulates how HIV cells are reproduced and destroyed. Most of the JavaScript was written by @anderspitman and the idea started with @aahowell3; we used PhaserIO for the game engine and I shamelessly took one of the PhaserIO Worley shader examples to use as a background.
  4. This website
  5. A color picking game This is a color picking game built using "Dear Imgui", so it's a gui-game. gooey game. It runs on Android and there's a release APK on GitHub.
  6. a project for using fun cmake features Some of the features include: external dependencie management, file checksum, and creating a shared/static library. I've been down the bottle of every build system, yeah, yeah. Smile softly, user story. yeah This is how I remind me how to use CMake the way I really am. Are we having fun yet? yeah, yeah
  7. Dear ImGui Fork Working on adding an Android example for "Dear Imgui"
  8. DAWG: DNA Analysis with Gaps Fork Working on @reedcarwright 's' DNA sequence simulator by adding CPP11 features and Boost tests.
  9. HeartWare Another school project, this one is an Android front-end for a fitness-driven with a social twist. It lets you query nearby gyms and post to FaceBook, presents data from FitBit devices on your Android device, and does data analysis using "Answer Set Programming".
